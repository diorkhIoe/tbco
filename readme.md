# SlashCommands Bot

Github post is a discord bot handler with slashcommands in v13

## How to install ?

You need to have Node.JS 16+
``````bash
git clone https://github.com/Xiaotoxdev/slashCommands-Bot.git
npm i
``````

## How to config ?

```.env
.env :
token=TOKEN
```

## How to start ?
```bash
node src/index.js # To start the bot
```
